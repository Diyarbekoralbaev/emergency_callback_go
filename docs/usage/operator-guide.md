# Руководство оператора

Операторам доступно ограниченное подмножество системы: они могут **создавать обратные вызовы** и
**просматривать список обратных вызовов**. Разделы только для администраторов (дашборд, оценки, бригады,
регионы, экспорт) недоступны — при попытке их открыть происходит перенаправление обратно на
список обратных вызовов.

## Вход в систему

Перейдите на `/users/login/` и войдите со своими учётными данными оператора (созданными
администратором через CLI). При успешном входе вы попадаете на **список обратных вызовов**.

## Создание обратного вызова

1. Откройте **Новый вызов** (`/callbacks/create/`).
2. Выберите **регион**, затем **бригаду**.
3. Введите номер телефона в формате `+998901234567`.
4. Отправьте — worker немедленно начинает дозвон.

## Просмотр обратных вызовов

**Все вызовы** (`/callbacks/`) отображает недавние обратные вызовы со статусом, бригадой, оценкой
(если есть), временем и длительностью. Фильтры: статус, бригада, регион, поиск, диапазон дат.
Откройте любую строку, чтобы перейти на страницу с подробностями.

## Что вам недоступно

| Действие | Разрешено? |
|--------|----------|
| Создание обратного вызова | ✅ |
| Просмотр списка и деталей обратных вызовов | ✅ |
| Дашборд / аналитика | ❌ только администратор |
| Аналитика оценок | ❌ только администратор |
| Управление бригадами / регионами | ❌ только администратор |
| Экспорт в Excel | ❌ только администратор |

Если вам нужна какая-либо из административных функций, обратитесь к администратору. Назначение роли
происходит при создании учётной записи (`createuser … operator`); изменить её может только администратор
с доступом к серверу.
