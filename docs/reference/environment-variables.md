# Переменные окружения

Вся конфигурация берётся из файла `.env` (или реальных переменных окружения),
читаемого при запуске. Пояснения и эффекты приведены в разделе
[Конфигурация](../getting-started/configuration.md); это краткая таблица.

!!! warning
    Значения задаются в виде простого `KEY=value`. Символы `<`, `>` или пробелы
    без кавычек ломают парсер dotenv и приводят к ошибке `required env var ... not set` при запуске.

## База данных

| Переменная | Обязательна | По умолчанию | Описание |
|----------|----------|---------|-------------|
| `DATABASE_URL` | да | — | DSN для PostgreSQL, например `postgres://ecb:pass@127.0.0.1:5432/emergency_callback?sslmode=disable`. |
| `DB_POOL_MAX_CONNS` | нет | `100` | Максимум соединений в пуле. |
| `DB_POOL_MIN_CONNS` | нет | `10` | Минимум соединений в пуле. |

## HTTP-сервер

| Переменная | Обязательна | По умолчанию | Описание |
|----------|----------|---------|-------------|
| `HTTP_ADDR` | нет | `:8000` | Адрес прослушивания. За прокси используйте `127.0.0.1:8000`. |
| `SITE_DOMAIN` | да | — | Публичный базовый URL; используется для построения ссылок голосования в SMS. Без завершающего слэша. |
| `SESSION_SECRET` | да | — | Ключ шифрования cookie сессии (32+ байт). Его изменение разлогинивает всех. |
| `CSRF_KEY` | да | — | Ключ CSRF; должен быть **ровно 32 байта** (`openssl rand -base64 24`). |

## Asterisk AMI

| Переменная | Обязательна | По умолчанию | Описание |
|----------|----------|---------|-------------|
| `AMI_HOST` | да | — | Хост AMI (часто `127.0.0.1`). |
| `AMI_PORT` | нет | `5038` | Порт AMI. |
| `AMI_USERNAME` | да | — | Пользователь AMI (должен существовать в Asterisk). |
| `AMI_SECRET` | да | — | Секрет AMI (должен совпадать с Asterisk). |
| `AMI_CALLER_ID` | нет | — | Номер Caller ID, отображаемый при исходящих вызовах. Только голый номер/строка. |
| `AMI_OPERATOR_QUEUE` | нет | `777` | Подсказка для назначения оператора. |
| `AMI_CALL_TIMEOUT` | нет | `60` | Секунды до отказа от вызова. |
| `AMI_RATING_RETRY_LIMIT` | нет | `3` | Допустимое число неверных нажатий клавиш до отказа. |
| `AMI_RATING_TIMEOUT` | нет | `10` | Секунды ожидания ввода оценки. |

## Eskiz SMS

| Переменная | Обязательна | По умолчанию | Описание |
|----------|----------|---------|-------------|
| `ESKIZ_EMAIL` | для SMS | — | Email учётной записи Eskiz. |
| `ESKIZ_PASSWORD` | для SMS | — | Пароль учётной записи Eskiz. |
| `ESKIZ_BASE_URL` | нет | `https://notify.eskiz.uz/api` | Базовый URL API. |
| `ESKIZ_DRY_RUN` | нет | `false` | При `true` SMS логируются вместо отправки. |

## Workers

| Переменная | Обязательна | По умолчанию | Описание |
|----------|----------|---------|-------------|
| `RIVER_MAX_WORKERS` | нет | `5` | Максимум одновременных фоновых задач (активных вызовов). |

## На этапе сборки / особые

| Переменная | Где | Описание |
|----------|-------|-------------|
| `MIGRATIONS_DIR` | подкоманда `migrate` | Переопределяет каталог миграций (по умолчанию `migrations`). |
| `ENABLE_PDF_EXPORT` | сборка документации | Установите в `1`, чтобы также рендерить сайт документации в PDF (`mkdocs build`). |
